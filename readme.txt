# 🏏 Hand Cricket Console Game

Welcome to the **Hand Cricket Console Game**!  
This game allows users to create accounts, login, reset passwords, and play a virtual hand cricket match against the computer — all from the command line using Python and SQLite.

---

## 📦 Features

- 👤 **User Registration** (with password confirmation and email check)
- 🔐 **Secure Login System**
- ❓ **Forgot Password via Security Question**
- 🏆 **High Score & Win Tracker (Stored in SQLite DB)**
- ⚾ **Batting/Bowling Toss System**
- 🔁 **Super Over in Case of Draw**
- 🎮 **Multiple Game Modes (User Bat First / Bowl First / Computer Plays)**
